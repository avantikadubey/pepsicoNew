# Getting Started

Planogram application 

This utilizes the following themes
- @planogram/core
- @planogram/design-system

## Setup

```
yarn
```

## Run Storybook

```
yarn workspace @planogram/design-system storybook
```

## Run App

```
yarn workspace app develop
```

## Build App

qa build
```
yarn workspace app build
```
